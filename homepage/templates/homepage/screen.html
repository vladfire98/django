Screen Linux

Создать сессию

    screen -S name-session

Свернуть сессию

    1) ctrl + a
    2) нажать "d"

Посмотреть активные сессии

    screen -ls

Вывод покажет id.имя-сессии

    vlad@ubuntu-hp:~$ screen -ls
    There is a screen on:
        13564.name-session	(11.09.2023 11:30:19)	(Detached)
    1 Socket in /run/screen/S-vlad.

Вернуться в сессию

    screen -r id

Если активен только один скрин, то можно использовать screen -r.
После флага -r можно указать id либо имя сессии.

Убить сессию

    1) ctrl + a
    2) k
