Tar

Создание архива
    tar -czvf new_name.tar.gz name_folder

чтобы сохранить архив в другом месте, можно воспользоваться флагом -C.

В архивируемые файлы можно передать несколько файлов

    tar -czvf archive.tar.gz file1 file2 file3 -C ~/

Параметры команды:

  • -c - создать новый архив
  • -z - использовать gzip для сжатия
  • -v - отображать подробную информацию о процессе архивации
  • -f - указывает имя файла для архива
Просмотр содержимого
    tar -tzvf archive.tar.gz

Параметры команды:

  • -t - отобразить список файлов в архиве
  • -z - использовать gzip для распаковки архива
  • -v - отображать подробную информацию о процессе распаковки
  • -f - указывает имя файла для распаковки
Распаковка архива tar.gz
    tar -xzvf file_name.tar.gz

Параметры команды:

  • -x - извлечь файлы из архива
  • -z - использовать gzip для распаковки архива
  • -v - отображать подробную информацию о процессе распаковки
  • -f - указывает имя файла для распаковки
